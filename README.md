## Paste Vault — Personal Knowledge Pipeline

One keystroke captures your clipboard and saves it to two destinations simultaneously:

- **Notion Paste Vault** — creates a new timestamped card in a Notion database via API
- **Local file** — appends to `~/Desktop/clips.txt` with timestamp
